## 0. 公開官網 (Official Landing Page - Pre-login)
**目標**：吸引新用戶 (老師/中心) 並展示產品價值。

| 模組 | 功能名稱 | 描述與備註 |
|:--- |:--- |:--- |
| **Branding** | **WOW Hero Section** | 高質感漸層背景，展示產品核心價值 (Slogan)。 |
| **Demo** | **互動排課沙盒 (Sandbox)** | **免登入** 體驗拖拉排課與衝突偵測 (紅/橘/靛藍動畫)。 |
| **Marketing** | **特色展示 (Showcase)** | 展示精美課表圖片、智慧媒合介面等特色。 |

## 1. 前台：老師 / 個人版 (Frontend - Teacher/Personal App)
**平台核心**：Mobile Web (RWD)，強調操作便利性與個人化管理。

| 模組 | 功能名稱 | 描述與備註 |
|:--- |:--- |:--- |
| **Auth** | **LINE 登入** | 核心登入方式。未註冊者自動建立帳號。 |
|  | **多中心綜合視圖** | 預設顯示 **所有已加入中心** 的總覽課表，並以顏色/標籤區分中心來源。 |
| **課表檢視** | **週行事曆 (RWD)** | 桌機顯示 7 天；手機自動切換 3 天視圖。**支援左右切換週次**。 |
|  | 課表細節查看 | 點擊課程卡片，查看教室、學生人數(選配)、課程筆記。 |
|  | **教學紀錄** | 針對單堂課程撰寫 **上課紀錄** 與 **備課筆記**。 |
|  | **匯出課表圖片** | 產生精美週課表圖片 (加載 **個人品牌標籤 3-5 個**)，供下載/分享。 |
| **個人行程** | **新增個人行程** | 免費版功能。在空檔新增私人事件 (非中心課程)。 |
|  | **循環行程設定** | 支援：單次、每天、每週、自訂 (如每2週) 的重複規則。 |
|  | 編輯/刪除行程 | 修改時間、標題或刪除 (支援刪除單次或整組循環)。 |
| **專業職能** | **專業檔案 (Profile)** | 設定 Bio 與 **多項技能 (Skills)**，每項技能可獨立掛載 **Hashtags** (如 瑜伽 > #空中瑜伽)。 |
|  | **求職/媒合設定** | 開關 **是否開放被搜尋**，並設定 **公開聯繫方式** (Line/Email)。開啟後中心可閱覽您的證照。 |
|  | **證照管理 (Cert)** | 上傳專業證照/檢定證明圖片。僅授權之中心可見。 |
| **中心排課** | 接收派課通知 | 當中心排入課程時，即時顯示於課表。 |
|  | **提出變更 (Exception)**| 針對中心課程提出：**停課 (Cancel)** 或 **改期 (Reschedule)**。 |
|  | **異動截止鎖定** | 視中心政策，超過截止日 (如開課前 7 天) 老師不可自行變更，需聯繫管理員。 |
|  | 衝突預檢 (Validate) | 提出改期時，系統即時檢查是否與已有課程衝突 (Overlap/Buffer)。 |
|  | 變更歷程 | 查看已提交的變更申請狀態 (待審、已核准、被拒絕)。 |
| **通知系統** | **明日課程提醒** | (New) 每日晚上自動推播 LINE 通知，預告隔日行程。 |
|  | **邀請與媒合通知** | (New) 收到中心邀請時，顯示對方提供的聯絡資訊。 |
|  | **異動緊急通知** | (New) 當課程被強制改期或取消時，即時推播警示。 |
| **設定** | 個人資料設定 | 修改姓名、大頭貼、**服務縣市/行政區**。 |
|  | 綁定中心 | 掃描 QR Code 或輸入邀請碼加入中心 (Membership)。 |

## 2. 後台：中心管理員 (Backend - Center Admin)
**平台核心**：Desktop Web，強調治理 (Governance) 與大量排課效率。

| 模組 | 功能名稱 | 描述與備註 |
|:--- |:--- |:--- |
| **Auth** | 管理員登入 | Email / Password 登入。可選配 LINE 綁定登入。 |
| **資源管理** | 老師管理 | 邀請老師、設定老師基本費率(選配)、停權老師。 |
|  | **內部評價與備註** | (New) 中心私有的評價 (1-5星) 與備註，僅該中心管理員可閱覽，不對老師公開。 |
|  | **老師證照檢視** | 檢視已加入老師的專業證照與履歷 Bio。 |
|  | 教室管理 | 設定教室名稱、容納人數、是否啟用。 |
|  | 課程 (Course) | 定義課程模板 (名稱、預設時長、顏色、**課程緩衝時間**)。 |
|  | 班別 (Offering) | 實際開課單位 (綁定 Course、老師、教室)，可設定 **是否允許緩衝覆寫**。 |
| **排課治理** | **政策設定 (Policy)** | 設定中心全域政策 (如方案限制)。緩衝時間則由 **課程模板** 獨立設定。 |
|  | **模板排課 (Grid)** | 視覺化週課表排程工具。支援拖拉操作。 |
|  | **週期分段 (Phase)** | (New) 支持 1-2 月與 3-4 月採用不同排課規則。支持規則生效區間設定。 |
|  | **國定假日管理** | (New) 自定義中心假日。設定後，該日週期課程自動設為「停課」不需手動操作。 |
|  | **批量驗證 (Bulk Validate)**| 儲存時對整週課程進行衝突檢查，回傳成功/失敗/待審清單。 |
|  | 強制排課 (Override) | 針對 Buffer 衝突，管理員可勾選「強制排入」並填寫原因。 |
|  | **智慧代課 (Smart Match)** | 點擊空檔或請假時段，根據 **技能** 與 **狀態** 自動推薦合適老師。 |
|  | **人才庫搜尋 (Talent Search)** | 主動搜尋已開啟「求職媒合」的老師。可用關鍵字或技能篩選。 |
| **審核中心** | **變更申請審核** | 列表顯示老師提出的 Cancel/Reschedule 申請。 |
|  | 審核決策 | 同意 (Approve) 或 拒絕 (Reject)。同意時需 Re-validate 確保無新衝突。 |
| **帳戶與權限** | 中心設定 | 修改中心名稱、方案升級/續費管理。 |
|  | **管理員帳號管理** | (New) 列出、新增、停用中心管理員。支援角色分級 (Owner/Admin/Staff)。 |
| **報表與紀錄** | **操作紀錄 (Audit Log)**| 查詢「哪個管理員 (actor) 在何時 修改了什麼」，確保責任追溯。 |
|  | 用量報表 (Plan) | 查看目前老師數、排課時數是否接近方案上限。 |

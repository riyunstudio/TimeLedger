# 01_SCOPE_AND_MILESTONES.md

## Must-have
- Teacher：personal sessions CRUD、schedule view
- Admin：centers、memberships、offerings、rules、validate、exceptions approve/reject
- Trial/Plan gating：30 天試用、到期唯讀、方案限制老師數
- Audit logs + event logs

## Out of scope
- Students / payment / LMS / App
- auto scheduling
- buffers (room/teacher)

## 1-week plan (suggested)
Day1 DB + teacher
Day2 center/membership/offering
Day3 rules + validate
Day4 exceptions + schedule view
Day5 gating + events + basic RWD
Day6 tests
Day7 internal pilot
